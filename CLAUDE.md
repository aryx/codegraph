# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Build Commands

```bash
./configure          # install opam deps + tree-sitter (first time / after dep changes)
make                 # build codegraph and codegraph_build executables
make test            # run all tests (dune runtest + tests/ integration tests)
make clean           # dune clean
make install         # dune install codegraph
```

Single test run: `dune runtest` for unit tests, or `make -C tests/cmt test` for the CMT integration test.

## Architecture

Codegraph is a dependency graph analyzer and visualizer using a Dependency Structure Matrix (DSM). It consists of two executables and supporting libraries.

### Executables

- **`codegraph`** (`src/main/Main.ml`) — GTK2/Cairo GUI viewer. Loads a `graph_code.marshall` file and renders an interactive DSM with drill-down from packages → modules → functions.
- **`codegraph_build`** (`src/builder/Main_build.ml`) — Builds the `graph_code.marshall` dependency database from source code. Supports multiple languages via `-lang` flag (cmt, c, java, php, js, ml, lisp, etc.).

### Core Data Structures (in `semgrep-pfff-langs/indexing/graph_code/`)

- **`Graph_code.t`** — Hypergraph of code entities. Nodes are `(name, Entity_code.kind)` pairs. Two edge types: `Has` (containment/hierarchy) and `Use` (dependency).
- **`Dependencies_matrix_code.dm`** — The DSM built from a graph via partitioning. Used by the viewer for rendering.
- **`Graph_code_opti.graph`** — Optimized graph for large codebases, cached as `.opti` files.

### Key Libraries

- **`src/viewer/`** — Matrix rendering (View_matrix.ml), MVC model (Model.ml), overlays, Cairo helpers.
- **`libs/gui/`** — GTK/lablgtk utility bindings.
- **`languages/GENERIC/`** — Language-agnostic pipeline:
  - `finding/` — file discovery by language
  - `parsing/` — dispatch to semgrep parsers → AST_generic
  - `linking/` — name resolution and graph building from AST_generic (`Graph_code_AST.ml`)

### Vendored Submodules

- **`semgrep-pfff-libs/`** — Shared utility libraries (commons, paths, profiling, visualization, etc.)
- **`semgrep-pfff-langs/`** — Language parsers (tree-sitter based), AST definitions, and indexing infrastructure (graph_code, database_code, highlight_code)

## Key Files

- `configure` — checks prerequisites, inits submodules, installs opam deps, sets up tree-sitter
- `tree-sitter-config.mk` — generated by configure; sets `TREESITTER_INCDIR`/`TREESITTER_LIBDIR` for dune
- `graph_code.marshall` — binary dependency database (output of codegraph_build, input of codegraph)
- `info.txt` — optional file in project root to constrain DSM partitioning order

## Testing

- Primary integration test: `tests/cmt/` — builds a small OCaml project, runs `codegraph_build -lang cmt`, verifies the graph
